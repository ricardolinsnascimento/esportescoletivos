Como Funciona o Esporte

O voleibol é um esporte coletivo, formado por 6 jogadores. Cada equipe possui um técnico. Esses 6 jogadores são posicionados na quadra e exercem uma determinada função. No vôlei, as funções são: recepção, levantador, atacante e bloqueio. Algumas regras do voleibol são:

  • Uma partida possui 5 sets;
  • Não existe tempo pré-determinado para cada set;
  • Cada set tem um máximo de 25 pontos com uma diferença mínima de 2 pontos;
  • Em caso de empate no set no final, a partida continua até que a diferença de dois pontos seja atingida (26 x 24, 27 x 25, etc.);
  • Após o saque, a equipe pode tocar somente três vezes na bola;
  • Ganha a equipe que vencer três sets;
  • Se houver empate nos sets (2x2) o 5º set será decisivo.
